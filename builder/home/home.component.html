Welcome to Application Builder

Application Builder is an open-source tool for you to create web applications in a no-code environment. It's being managed by the Software AG Global Presales team but not officially supported by Software AG. You can find the source code on Software AG Open Source.


Application Builder also allows you to,

Re-brand the applications.

Simulate devices.

Install runtime widgets.
